# 相關介紹
### 目的
主要用於查詢書籍，可以藉由書籍名稱、作者的資訊、以及書籍的類別等等來查詢想了解的書籍。  
### 過程及方法
設計一個使用者介面，以及所需的功能（輸入、下拉式選項、勾選等等），根據所輸入的內容用資料庫進行相對應的分類。主要使用了 html、PHP、MySQL 與 phpMyAdmin 來實作書籍查詢系統。  
### 使用技術
- PHP
- MySQL
